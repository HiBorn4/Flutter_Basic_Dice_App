## Getting Started

1. Clone this repository.
   ```sh
   git clone https://github.com/HiBorn4/Alumni_Connect
   ```

2. Change directory to your project folder.
   ```sh
   cd Alumni-Connect
   ```

3. Install the required dependencies.
   ```sh
   flutter pub get
   ```
   
4. Run your app:
   ```sh
   flutter run
   ```
